운영 원칙

  • public 화면은 제품 언어만 유지 운영 정보는 내부 화면에서만 봅니다.

현재 범위

  1. public 화면과 운영 화면의 정보 구조를 분리했다.
  2. 추천 만족도 집계를 `/dashboard/recommendation/feedback`으로 옮겼다.
  3. Dashboard 화면은 `ADMIN` role 세션으로만 접근 가능하게 보호한다.
